# SmartLearn

SmartLearn is a full-stack MERN-based academic portal designed to streamline academic management and collaboration.

## Features
- Role-based authentication and authorization
- JWT, OTP, and bcrypt secured authentication
- Attendance and assignment management
- Exam scheduling and timetable system
- Virtual study rooms with video meeting support
- Real-time messaging and collaboration features
- Study material and resource management

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.io
- WebRTC
